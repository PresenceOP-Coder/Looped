# Support

## Where to Ask for Help

- Usage questions: open a GitHub Discussion or issue tagged `question`
- Bug reports: open a GitHub issue using the bug report template
- Feature ideas: open a GitHub issue using the feature request template

## Before Opening an Issue

1. Search existing issues first.
2. Include your platform and Flutter/Dart versions.
3. Provide clear steps to reproduce and expected behavior.

## Security Issues

For vulnerabilities or exposed secrets, follow [SECURITY.md](SECURITY.md).
